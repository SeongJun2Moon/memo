### 2. 설정
- 첫 설치시 'Do not import settings'선택
- env
  - Configs → Settings → Project interpreter → System Interpreter → Python.exe경로 설정
- venv
  - PyCharm-> Preferences -> Project -> Python Interpreter -> 인터프린터 선택 중 Show All → add(+)
